# E-commerce Sales Performance Analyzer 📈

## Project Overview

This project simulates a retail data analysis workflow. It focuses on processing large-scale transactional data to identify top-performing products and regional trends using Python's data science stack.

## Features

- Automated Data Generation: Uses NumPy for high-performance synthetic data creation.
- Advanced Filtering: Implements Boolean indexing for multi-condition data slicing.
- Statistical Analysis: Calculates custom performance metrics using standard deviation and mean imputation for missing values.
- Interactive Summaries: Generates multi-dimensional pivot tables with grand totals.

## Tech Stack

- Python
- Pandas (Data Manipulation)
- NumPy (Mathematical Operations)

## How to Run

Clone this repository: `https://github.com/proshanta-pal/e-commerce-store-data-analyzer.git`

Install dependencies: `pip install pandas numpy`

Run the Jupyter Notebook or Python script to generate the `top_10.csv` report.
